# logisat-warsaw

[![codecov](https://codecov.io/gh/przemekwrona/logisat-warsaw/graph/badge.svg?token=DQOLK347TY)](https://codecov.io/gh/przemekwrona/logisat-warsaw)

## Build with Docker

Build project with command

```bash
$ npm install
```

Setup database component and backedn with

```bash
$ docker compose up
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
